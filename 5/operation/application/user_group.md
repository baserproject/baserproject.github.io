# ユーザーグループ管理

管理画面の機能制限をグループ単位で設定・管理するための機能です。
「ブログ編集のみ不可」「全機能利用不可」など役割に応じた権利を与え、
誤操作防止やセキュリティ向上を図れます。

## 新規ユーザーグループ登録

ユーザーグループ一覧の左上「新規追加」からユーザーグループの追加を行います。

### ユーザーグループ名
- システム側がグループを識別するために利用します
- 重複しない名前を半角で入力してください
- adminsグループは変更ができません

### 表示名
- ユーザー側がグループを識別するために利用します
- 識別しやすい名称を入力してください。特に制限はありません。

### その他
- コンテンツのドラッグ＆ドロップ移動機能を利用するを有効にした場合
  コンテンツ一覧のツリー構造で、ドラッグ＆ドロップでコンテンツの移動
  を許可します。

### 認証プレフィックス設定
- 管理システム
  - 管理画面へのアクセス権です。
    フルアクセスの場合、すべての機能が制限なく利用できます。
    限定アクセスの場合、特定の機能のみの制限がかかった状態です。
  - **制限は「[アクセスルールグループ管理](https://baserproject.github.io/5/operation/application/access_rule){:target="_blank"}」にて設定できます。**

- Admin Web API
  - 管理画面の各種機能を、Web API（プログラム）経由で実行できるようにするためのアクセス権限です。
    画面での手動操作ではなく、プログラムによる自動処理や、他のシステム・アプリとデータを連動させたい場合は、権限を有効にしてください。

## ユーザーグループ一覧画面

ユーザーグループの一覧を確認できます。
このページからユーザーグループに対してのユーザーグループの制限、ユーザーグループの編集、
ユーザーグループのコピー、ユーザーグループの削除ができます。

- [ユーザーグループの制限](https://baserproject.github.io/5/operation/application/access_rule){:target="_blank"} - baserCMSユーザーマニュアルを参照ください。

- ユーザーグループの編集
  - 登録したユーザーグループの編集ができます

- ユーザーグループのコピー
  - ユーザーグループのコピーが作成されます。
    ユーザーグループ名と表示名の末尾に「_copy」が付きます。

- ユーザーグループ削除
  - ユーザーグループ一覧から削除します。
   削除したいユーザーグループにユーザーが所属している場合は削除できません。
   またユーザーグループを削除すると、削除したユーザーグループに設定されていたアクセスルールグループも全て削除されます。
