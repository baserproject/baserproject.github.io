# ユーザーグループ管理

管理画面の機能制限をグループ単位で設定・管理するための機能です。
ユーザーをグループに登録することで、「ブログ編集のみ不可」「全機能利用不可」など役割に応じた権利を与え、誤操作防止やセキュリティ向上を図れます。

## 新規ユーザーグループ登録

ユーザーグループ一覧の左上「新規追加」からユーザーグループの追加を行います。

### ユーザーグループ名
- 重複しない名前を半角で入力してください
- adminsの場合は変更ができません

### 表示名
- 識別しやすい名称を入力してください。特に制限はありません。

### その他
- コンテンツのドラッグ＆ドロップ移動機能を利用するを有効にした場合
  コンテンツ一覧のツリー構造で、ドラッグ＆ドロップでコンテンツの移動
  を許可します。

### 認証プレフィックス設定
- 管理システム
  - 管理画面へのアクセス権です。
  フルアクセスの場合、すべての機能が制限なく利用できます。
  限定アクセスの場合、特定の機能のみの制限がかかった状態です。

- Admin Web API
  - プログラム自動操作に対してのアクセス権です。
  外部ツールと連携させたい場合、有効にしてください。

## ユーザーグループ一覧画面

ユーザーグループの一覧を確認できます。このページからユーザーグループに対してのアクセスルールの設定、ユーザーグループの編集、ユーザーグループのコピー、ユーザーグループの削除ができます。

### アクセスルールの設定
- [アクセスルール管理について](https://baserproject.github.io/5/operation/application/access_rule){:target="_blank"} - baserCMSユーザーマニュアルを参照ください。

- ユーザーグループの編集
  - 登録したユーザーグループの編集ができます

- ユーザーグループのコピー
  - ユーザーグループのコピーが作成されます。
  ユーザーグループ名と表示名の末尾に「_copy」が付きます。

- ユーザーグループ削除
  - ユーザーグループ一覧から削除します。
  削除する場合、そのグループに関連するユーザーは削除されませんが、関連するアクセスルールはすべて削除されます。関連するユーザーは管理者グループに所属することになります。
